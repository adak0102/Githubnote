# 마크다운

## I. 마크다운이란?

문법이 쉬운 **마크업** 언어

- 마크업: *내용을 구조화* 하는 표시 (문서내의 역할을 표시)

```python
def hello():
    return 'hello'

hello()

if age>20:
    print('아재')
else:
    print('어린이')
```

## II. 마크다운 문법

### (1)`#` 헤더

``` markdown
# 이것은 대제목
## 이것은 중제목
### 이것은 소제목
#### 이것은 부제목
```

### (2) `-` ,`*`  뷸렛포인트

- 뷸렛 1

  - 뷸렛2

    - 뷸렛3

      ```markdown
      - 뷸렛1 
      	-뷸렛1
      		-뷸렛3
      ```

      

### `1.` 넘버링

1. 넘버링1
   1. 넘버링2
      1. 넘버링3

```markdown
-넘버링1
	-넘버링2
		-넘버링3
```



### (4) `>` 인용구

> 이것은 인용구입니다.
